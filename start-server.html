🏈 ローカルサーバー起動方法

⚠️ 重要: お問い合わせフォームをテストするには、PHPが動作するローカルサーバーが必要です。

🚀 簡単な起動方法

方法1: PHPの内蔵サーバーを使用（推奨）

  1. ターミナル（コマンドプロンプト）を開く
  2. プロジェクトフォルダに移動:
cd /Users/yusukeyoshida/coding/claude/football-team
  1. PHPサーバーを起動:
php -S localhost:8000
  1. ブラウザで以下にアクセス:
http://localhost:8000

方法2: Python 3を使用

  1. ターミナルでプロジェクトフォルダに移動
  2. Pythonサーバーを起動:
python3 -m http.server 8000
注意: Pythonサーバーの場合、PHPが動作しないため、お問い合わせフォームは「Failed to fetch」エラーになります。

方法3: Node.js (live-server)を使用

  1. Node.jsとnpmがインストール済みの場合:
npx live-server --port=8000

🔧 トラブルシューティング

エラー「Failed to fetch」が出る場合

  • PHPサーバーが起動していることを確認
  • ブラウザの開発者ツール（F12）でコンソールエラーを確認
  • ファイルパスが正しいか確認

PHPがインストールされていない場合

  • Mac: Homebrewでインストール: brew install php
  • Windows: PHP公式サイトからダウンロード
  • Linux: パッケージマネージャーでインストール: sudo apt install php

✅ 動作確認方法

正常に動作している場合:
  1. ホームページが表示される
  2. ナビゲーションメニューが動作する
  3. お問い合わせフォームから送信できる
  4. 「送信完了（デモ版）」のモーダルが表示される
